移动云手机开启ADB

2024.07.29 教程
这篇文章上次修改于 696 天前，可能其部分内容已经发生变化，如有疑问可询问作者。

准备工作

首先下载开启 ADB 的 shell 脚本和甲壳虫助手，点击下载。

对了前提得是你得有 ROOT 权限
获取 ROOT 权限的方法可以参考移动云手机获取 ROOT 权限

开始操作

  1. 打开移动云手机，进入 MT 管理器，将下载下来的脚本放到 data 目录下，属性给 777 权限，执行的时候选择拓展包环境和 ROOT 权限执行。

  1. 执行成功后会提示 IP 和端口，需要重启云手机生效。

  2. 在本地安装甲壳虫助手，输入上面的 ip:端口 连接。

重要提示

开启了 ADB 后是没有任何防护和验证的，别人用你的 ip:端口就可以连接你的手机，所以一定要保管好你的 IP 和端口，不要泄露出去。

而且云手机里面是可以一键登录移动云盘的，有个人隐私或者重要资料的请小心被盗.

不显示拓展环境的请点击 MT 管理器侧边的终端模拟器进行下载就显示了.
